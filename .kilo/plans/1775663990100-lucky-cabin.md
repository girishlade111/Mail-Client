### 5. Unified Inbox Mode

**In MailContext.jsx:**
```javascript
const unifiedEmails = useMemo(() => {
  if (unifiedInboxEnabled) {
    return allEmails.map(e => ({ ...e, accountId: e.accountId }));
  }
  return allEmails.filter(e => e.accountId === activeAccountId);
}, [unifiedInboxEnabled, activeAccountId, allEmails]);
```
