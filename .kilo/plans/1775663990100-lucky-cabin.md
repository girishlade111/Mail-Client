## Implementation Plan

### 1. Enhanced Keyboard Shortcuts

**Add to App.jsx handleKeyDown:**
```javascript
// Navigation
'g i' -> Go to Inbox
'g s' -> Go to Starred  
'g d' -> Go to Drafts
'g t' -> Go to Sent
'g a' -> Go to All Mail
'g c' -> Go to Contacts

// Message Actions
'c' -> Compose new email
'r' -> Reply to sender
'a' -> Reply all
'f' -> Forward
'e' -> Archive
'#' -> Delete (or number key)
'!' -> Mark important
's' -> Toggle star
'u' -> Mark unread

// Selection
'x' -> Select current
'* a' -> Select all
'* n' -> Deselect all

// Interface
'/' -> Focus search
'?' -> Show shortcuts help
```
